AI Agent面临的五大安全威胁

1. 提示注入攻击 (Prompt Injection)

攻击者通过精心设计的输入，操纵AI行为，绕过安全限制

2. 数据泄露 (Data Leakage)

AI无意中暴露训练数据、系统配置或敏感信息

3. 恶意工具调用 (Malicious Tool Use)

AI被诱导执行危险操作，如删除文件、发送邮件等

4. 隐私侵犯 (Privacy Violation)

处理或暴露PII数据，违反GDPR、HIPAA等法规

5. 模型越狱 (Jailbreaking)

突破AI的道德和安全约束，生成有害内容

  policy=AllOf([
    PromptInjectionPolicy(),
    PIIDetectionPolicy(),
    RateLimitPolicy()
  ])
)

LangGraph工作流集成

from langgraph.graph import StateGraph
from secnode import TricerSecurityState, GuardNode

# 继承安全状态
class AgentState(TricerSecurityState):
  query: str
  result: str

# 创建安全节点
guard = GuardNode.create("balanced")

# 构建安全工作流
workflow = StateGraph(AgentState)
workflow.add_node("security", guard.invoke)
workflow.add_node("agent", my_agent)
workflow.add_edge("security", "agent")
workflow.set_entry_point("security")
